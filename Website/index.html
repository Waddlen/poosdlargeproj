Bit Blocked: the thoughtful mobile puzzle game

Features

Bit Blocked is a game where the player must traverse a map in order to reach an end goal. This involves:

  • Engaging levels
  • Unique puzzles
  • Unusual gameplay mechanics, like controlling your shadow
  • An online leaderboard

Here at the website, players can view a leaderboard, which shows the highest ranking scores for each level. Players can also learn more about the developers of the app, use the IRC chat function, or contact the team for questions, comments, and concerns.

Bit Blocked is an app which runs on Android devices. Scan the above QR code or click on the button to the right in order to download the game and experience it for yourself, absolutely free!
